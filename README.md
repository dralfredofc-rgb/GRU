## Corrección crítica (v2.6)

N5≈3.5×10^18 eventos necesarios para 5σ (no 1 como se afirmaba en v2.5). GRU es esencialmente indetectable por LISA en su misión primaria con el mecanismo de screening actual (ver §F.5 del paper).

## Descripción

GRU (Geometría Radial Unitaria) es un marco fenomenológico que conecta la gravedad cuántica causal (CDT) con predicciones observables para LISA y pruebas del Sistema Solar.
