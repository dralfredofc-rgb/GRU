## Descripción

GRU (Geometría Radial Unitaria) es un marco fenomenológico que conecta la gravedad cuántica causal (CDT) con predicciones observables para LISA y pruebas del Sistema Solar.

## Características v2.5

- **Root Invariance Protocol (§A.46):** invariancia topológica del spine causal (ds=0.9602±0.0413)
- **fspine oficial:** 7.7912 mHz, ΔtGRU = 128.35 s
- **MDR framework:** desde el Hessiano de Regge (γ=0.0564, verificado a 0.31σ)
- **Computación cuántica:** canal CPTP de amortiguamiento (4/4 tests PASS)
- **Compatibilidad Sistema Solar:** γ_PPN = 1, ruptura SO(3)≈10^-82
- **Falsificabilidad LISA:** A_final = 1.46×10⁻¹⁶, detectable en N=1 evento SMBHB a 5σ
